# ESP32 Retro Phone

Прошивка для превращения советского дискового телефона (ТА-57, ВЭФ, Телта и др.) в полноценную Bluetooth-гарнитуру.

## Возможности

- Звонки через Bluetooth Classic (HFP) — соединяется со смартфоном как гарнитура
- Дисковый номеронабиратель — набор номера импульсами, тайм-аут 3 секунды после последней цифры
- Механический звонок — оригинальный колокольчик звонит при входящем вызове (28 В от MT3608)
- Гудок готовности 425 Гц — при поднятии трубки, если BT подключён
- Сигнал «занято» — если BT не подключён
- Питание от LiPo аккумулятора с зарядкой по USB (TP4056)
- Сброс привязки — набрать `000` на диске без активного BT-соединения

## Железо

| Компонент | Примечание |
|---|---|
| ESP32 DevKit (WROOM-32) | Обязательно оригинальный ESP32 — нужен Classic BT |
| WM8960 Audio Codec | SparkFun breakout или клон |
| MT3608 boost converter | Настроить на 28 В для звонка |
| TP4056 + LiPo 18650 | Питание |
| AMS1117-3.3 | LDO для ESP32 и WM8960 |
| 2× BS170 MOSFET | Драйвер катушек звонка |

Полная схема подключения и список компонентов — [ASSEMBLY.md](ASSEMBLY.md).

## Прошивка

Требует **ESP-IDF v5.2+**.

```bash
idf.py set-target esp32
idf.py build
idf.py -p COM3 flash monitor
```

## Структура проекта

```
main/
├── main.c          — точка входа, инициализация
├── config.h        — все GPIO и параметры
├── state_machine.c — конечный автомат телефона
├── audio.c         — I2S + I2C (WM8960), LEDC MCLK
├── bt_hfp.c        — Bluetooth Classic HFP
├── bell.c          — драйвер механического звонка
├── rotary_dial.c   — обработка импульсов диска
├── hook_switch.c   — рычажный переключатель
└── dial_tone.c     — генерация тонов (гудок, занято)
```

## Сопряжение с телефоном

1. Включить устройство
2. Настройки → Bluetooth → найти **"Retro Phone"**
3. PIN: `0000`
