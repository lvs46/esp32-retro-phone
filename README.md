# ESP32 Retro Phone — Спектр-3 (ТА-11320)

Прошивка для превращения советского дискового телефона **Спектр-3 (ТА-11320)** в полноценную Bluetooth-гарнитуру для смартфона.

## Возможности

- **Bluetooth Classic HFP** — звонки через смартфон, ESP32 виден как hands-free устройство
- **Дисковый набор** — импульсы реального диска с гейтингом по off-normal контакту; ошибки не пропускаются
- **Механический звонок** — оригинальная катушка МТ звонит при входящем вызове через H-мост (L293D, переменный ток 25 Гц от 28 В)
- **Гудок 425 Гц** — при поднятии трубки, если BT подключён
- **Сигнал «занято»** — если BT не подключён
- **Авто-переподключение** — после reset/power-on устройство само поднимает связь со спаренным телефоном
- **Питание от LiPo** с зарядкой по USB (TP4056)

## Сервисные коды (трубка лежит)

Набираются диском при положенной трубке:

| Код | Действие |
|---|---|
| **000** | Сброс всех BT pairing (с подтверждением — короткий звон) |
| **999** | Sweep-тест звонка по частотам 10–28 Гц для поиска механического резонанса |

## Железо

| Компонент | Примечание |
|---|---|
| ESP32 DevKit (WROOM-32) | Обязательно оригинальный ESP32 — нужен Classic BT |
| WM8960 Audio Codec | SparkFun breakout или клон |
| MT3608 boost converter | Настроить на 28 В для звонка |
| **L293D** | H-мост для драйва катушки МТ переменным током |
| TP4056 + LiPo 18650 | Питание |
| AMS1117-3.3 | LDO для ESP32 и WM8960 |
| Электретный капсюль | Замена угольного микрофона (МК-16, ~16 мм) |

Поляризованной катушке МТ Spektr-3 нужен **переменный** ток (постоянка только притягивает якорь, а звенит он от колебаний). H-мост L293D даёт ±25 В переменки от одного источника 28 В.

Полная схема подключения и калибровка — [ASSEMBLY.md](ASSEMBLY.md).

## Прошивка

Требует **ESP-IDF v5.2+**.

```bash
idf.py set-target esp32
idf.py build
idf.py -p COM3 flash monitor
```

## Структура проекта

```
main/
├── main.c          — точка входа, инициализация
├── config.h        — все GPIO и параметры
├── state_machine.c — конечный автомат телефона + сервис-коды
├── audio.c         — I2S + I2C (WM8960), LEDC MCLK
├── bt_hfp.c        — Bluetooth Classic HFP + авто-реконнект
├── bell.c          — драйвер звонка (L293D, sweep-тест)
├── rotary_dial.c   — обработка диска (ИК + ЗП, гистерезис)
├── hook_switch.c   — рычажный переключатель
└── dial_tone.c     — генерация тонов (гудок, занято)
```

## Сопряжение с телефоном

1. Включить устройство
2. Настройки → Bluetooth → найти **"Retro Phone"**
3. PIN: `0000`

ESP32 запоминает телефон в NVS. При следующем включении подключение пройдёт автоматически через ~5–10 секунд.
